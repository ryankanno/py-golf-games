<!-- GETTING STARTED -->
## 🚀 Getting Started

### Prerequisites

* [poetry](https://python-poetry.org/docs/#installing-with-the-official-installer)
```sh
curl -sSL https://install.python-poetry.org | python3 -
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/ryankanno/py-golf-games.git
```
2. Install
```sh
poetry install
```

## 🛠️ Usage
